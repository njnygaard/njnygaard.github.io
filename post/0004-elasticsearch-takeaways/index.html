Elasticsearch Takeaways

28 Sep 2017, 16:27

I’ve been using Elasticsearch for my personal project for about a year now and I am enjoying the functionality. My project has an Angular 4 frontend and a Go backend. I enjoy the alignment between TypeScript and Go and how it lets me bring static(ish) typing to the front -> contract -> back of my application. I am using olivere’s elasticsearch client to write a good portion of the data storage as well. If “elasticsearch-as-a-datastore” is offensive to you, just stop reading ;)
